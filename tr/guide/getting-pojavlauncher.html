                                                    1. Jailbroken Installation

                                                    # Releases & GitHub Actions

                                                    1. You can get prebuilt app from stable releasesopen in new window or automatic buildsopen in new window.

                                                    # Google Play

                                                    1. You can get it from Google Play by clicking hereopen in new window or by clicking the badge below: open in new window

                                                    Note

                                                    PojavLauncher is removed from the shelves of Google Play. It is advised to use this GitHub Actionopen in new window

                                                    # Building from source

                                                    • Even though it is well-explained, building from source requires skill, internet and time.

                                                    • You can try building on Android with Termuxopen in new window, but no support will be included for it.

                                                    • It is recommended to clone the PojavLauncher repository before starting Setting Up JREopen in new window

                                                    • git clone https://github.com/PojavLauncherTeam/PojavLauncher

                                                    # Setting up JRE

                                                    • You can download it from CI Auto Buildsopen in new window if you don't want to/can't build.

                                                    • Source: PojavLauncherTeam/android-openjdk-build-multiarchopen in new window

                                                    # Android
                                                    • Clone the git repository of PojavLauncherTeam/android-openjdk-build-multiarch
                                                    git clone https://github.com/PojavLauncherTeam/android-openjdk-build-multiarch
                                                    • cd to the cloned repository
                                                    cd android-openjdk-build-multiarch
                                                    • Download Android NDK r10e from hereopen in new window and place it in this directory with a file explorer. (Can't automatically download because of EULA)

                                                    Note

                                                    Do not attempt to build using a newer or older NDK, it will lead to compilation errors.

                                                    # iOS
                                                    • Get the latest Xcodeopen in new window (tested with Xcode 12).

                                                    PlatformArchitectureTargetTarget_JDK
                                                    Androidarmv8/aarch64aarch64-linux-androidaarch64
                                                    Androidarmv7/aarch32arm-linux-androideabiarm
                                                    Androidx86/i686i686-linux-androidx86

                                                    iOS/iPadOSarmv8/aarch64aarch64-macos-iosaarch64
                                                    # Building JRE (Java Runtime Environment)

                                                    :::: code-group ::: code-group-item Android

                                                    export BUILD_IOS=0

                                                    ::: ::: code-group-item iOS

                                                    export BUILD_IOS=1

                                                    ::: ::::

                                                    export BUILD_FREETYPE_VERSION=[2.10.4]

                                                    Tips

                                                    You can use versions between 2.6.2 and 2.10.4 but 2.10.4 is preferred.

                                                    export JDK_DEBUG_LEVEL=[release]

                                                    Tips

                                                    You can use release, fastdebug and debug, but release is preferred

                                                    export JVM_VARIANTS=[client]

                                                    Tips

                                                    You can use client and server variants, Client is for aarch32 and server is for other architectures

                                                    (Android only)

                                                    ./extractndk.sh

                                                    ./maketoolchain.sh

                                                    • Get CUPS, Freetype and build Freetype

                                                    ./getlibs.sh

                                                    ./buildlibs.sh

                                                    • Clone JDK, run once

                                                    ./clonejdk.sh

                                                    • Configure JDK and build

                                                    ./buildjdk.sh

                                                    Tips

                                                    If no configuration is changed, run sh makejdkwithoutconfigure.sh instead

                                                    • And finally, pack the built JDK

                                                    ./removejdkdebuginfo.sh

                                                    ./tarjdk.sh

                                                    • Updating translations is easier thanks to language list updater scripts

                                                    • Just run this command and it'll be automatically updated from Crowdin.

                                                    :::: code-group ::: code-group-item Windows

                                                    scripts\languagelist_updater.bat

                                                    ::: ::: code-group-item Linux/MacOS

                                                    chmod +x scripts/languagelist_updater.sh
bash scripts/languagelist_updater.sh

                                                    ::: ::::

                                                    :::: code-group ::: code-group-item Windows

                                                    ./gradlew :jre_lwjgl3glfw:build

                                                    ::: ::: code-group-item Linux/MacOS

                                                    gradle :jre_lwjgl3glfw:build

                                                    ::: ::::

                                                    ::: ::: code-group-item Linux/MacOS

                                                    ./gradlew :app_pojavlauncher:assembleDebug
